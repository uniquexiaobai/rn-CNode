## 安装及运行：
##### 1. 下载
```bash
$ git clone git@github.com:uniquexiaobai/rn-CNode.git
```
##### 2. 安装依赖
```bash
$ cd rn-CNode && npm install
```
##### 3. 运行
- Android版运行：
```bash
$ react-native start
$ react-native run-android
```
